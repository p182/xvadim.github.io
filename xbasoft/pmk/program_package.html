Общие сведения

Программы для эмулятора хранятся в файлах с расширением pmk. Если в каталоге с фалом программы находится файл с таким же названием и расширением html эмулятор также загружает его и показывает при выборе пункту меню "Описание".

При подготовке описания программы можно использовать шаблон из архива. Это шаблон представляет собой html-файл с набором предопределенных css-стилей для красивого оформления описаний.
