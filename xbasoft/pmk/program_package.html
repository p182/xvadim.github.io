Общие сведения

Программы для эмулятора хранятся в файлах с расширением pmk.

Экспортировать содержимое памяти эмулятора в pmk-файл можно, выбрав меню "Экспорт".

Файл описания программы

Эмулятор также может в информационной панели (которая вызывается кнопкой (i) ) показывать описание программы. Для этого необходимо подготовить описание в html-формате, импортировать его с помощью пункта меню "Импорт описания" и потом нажать кнопку D (description - описание) в информационной панели.

При подготовке описания программы можно использовать шаблон из архива. Это шаблон представляет собой html-файл с набором предопределённых css-стилей для красивого оформления описаний.

Текстовый импорт

Эмулятор также может выполнить импорт специально оформленных текстовых файлов. Если у файла расширение mk61, то используется текстовый импорт.

Эмулятор понимает несколько способов оформления текстов программ:

  • Формат проекта Lord_BSS: 
    
#	|	00	01	02	03	04	05	06	07	08	09
00	|	В/О	Fx<o	05	Пх9	С/П	3	хПС	ПхД	Кинв	С/П
10	|	Fx=o	39	КПП8	Kx<oB	ПхЕ	С/П	Пх5	Пх2	–	хП0
  • Формат js-эмуляторов 1 и 2 с номерами команд: 
    
00.В/О  01.Fx<0 02.05   03.Пx9   04.С/П 05.3   06.xПC 07.ПxD 08.Kинв 09.С/П 
10.Fx=0 11.39   12.KПП8 13.Kx<0B 14.ПxE 15.С/П 16.Пx5 17.Пx2 18.-    19.xП0
  • ... и без номеров ("Простой код"): 
    
В/О  Fx<0 05   Пx9   С/П 3   xПC ПxD Kинв С/П 
Fx=0 39   KПП8 Kx<0B ПxE С/П Пx5 Пx2 -    xП0
