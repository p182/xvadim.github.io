Q: Why does μBarometer not collect data?

A: There are several reasons:

  • Doze mode - μBarometer can't collect data when a device is in Doze mode
  • Some kind of task killers/battery savers can kill μBarometer. Exclude μBarometer from the list of killed apps
  • Some tools like Zapper Task Killer, Clean Master, Greenify or any Antivirus software can kill μBarometer
